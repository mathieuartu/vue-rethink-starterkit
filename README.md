# vue-rethink-starterkit

## Project setup
```
yarn install
```

### Serves an API using RethinkDB
```
yarn api
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your tests
```
yarn test
```

### Lints and fixes files
```
yarn lint
```
